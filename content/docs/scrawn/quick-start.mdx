Get Scrawn backend and SDK running in your project. This guide will walk you through setting up the backend, configuring your database, and integrating the SDK.

## 1. Clone the Repository

<Tabs items={['HTTPS', 'Github CLI']}>
  <Tab value="HTTPS">
    ```bash
    git clone https://github.com/ScrawnDotDev/Scrawn.git
    cd Scrawn
    ```
  </Tab>
  <Tab value="Github CLI">
    ```bash
    gh repo clone ScrawnDotDev/Scrawn
    cd Scrawn
    ```
  </Tab>
</Tabs>

### 1.1 Install packages

<Tabs items={['bun', 'npm', 'yarn']}>

  <Tab value="yarn">

## 2. Configure User ID Type

Open `src/config/identifiers.ts` and set your user ID type based on your database schema:

```ts title="src/config/identifiers.ts"
// Choose the ID type that matches your database
export const USER_ID_CONFIG = ID_CONFIGS.uuid;  // [!code --]
export const USER_ID_CONFIG = ID_CONFIGS.uuid;  // For UUID user IDs // [!code ++]
// export const USER_ID_CONFIG = ID_CONFIGS.bigint;  // For BigInt user IDs // [!code ++]
// export const USER_ID_CONFIG = ID_CONFIGS.int;  // For Integer user IDs // [!code ++]
```

## 3. Setup Environment Variables

Copy the example environment file to create your local configuration:

<Tabs items={['macOS/Linux', 'Windows (CMD)', 'Windows (PowerShell)']}>
  <Tab value="macOS/Linux">
    ```bash
    cp .env.example .env.local
    ```
  </Tab>
  <Tab value="Windows (CMD)">
    ```cmd
    copy .env.example .env.local
    ```
  </Tab>
  <Tab value="Windows (PowerShell)">
    ```powershell
    Copy-Item .env.example .env.local
    ```
  </Tab>
</Tabs>

Now edit `.env.local` with your configuration:

```bash title=".env.local"
HMAC_SECRET="YOUR_VERY_SECRET_CODE_GOES_HERE" 
DATABASE_URL="YOUR_NOT_SO_SECRET_DATABASE_URL_GOES_HERE"
LEMON_SQUEEZY_API_KEY="" 
LEMON_SQUEEZY_STORE_ID=""
LEMON_SQUEEZY_VARIANT_ID="" 
```

### 3.1 Generate HMAC Secret

 Generate a secure HMAC secret:

```bash
openssl rand -base64 64
```

Copy the output and paste it as your `HMAC_SECRET` value.

### 3.2 Database URL

Set up a PostgreSQL database and paste the connection URL. You can use:
- [Supabase](https://supabase.com/) (free tier available)
- [Neon](https://neon.tech/) (serverless Postgres)
- [Railway](https://railway.app/) (easy deployment)

Example: `postgresql://user:password@host:5432/database`

### 3.3 Lemon Squeezy Configuration

1. Sign up at [Lemon Squeezy](https://lemonsqueezy.com/)
2. Follow their [Getting Started Guide](https://docs.lemonsqueezy.com/guides/developer-guide/getting-started)
3. Get your API key, Store ID, and Variant ID from the dashboard

<Callout type="warn" title="Important">

Copy the SQL `INSERT` statement from the output and run it in your database provider's SQL editor (Supabase SQL Editor, Neon Console, etc.).

## Enjoy! ðŸŽ‰

Start tracking usage and billing your customers:

```typescript
import { Scrawn } from '@scrawn/core';

const scrawn = new Scrawn({
  apiKey: process.env.SCRAWN_KEY, // [!code highlight]
  baseURL: process.env.SCRAWN_BASE_URL, // [!code highlight]
});

// Track usage
await scrawn.sdkCallEventConsumer({ 
  userId: '0a1e38a5-e2f3-450f-9796-59ad406d93dc', 
  debitAmount: 5000,
});
  
```

<Callout type="warn" title="Configuration">
**API Key:** Use the key you generated in [Step 5](#5-generate-initial-api-key) or get one from your [Dashboard](/docs/scrawn/dashboard-setup#creating-a-new-api-key).

**Base URL:** Use the URL you get after [deploying your Scrawn backend](#6-deploy-scrawn-backend).
</Callout>
