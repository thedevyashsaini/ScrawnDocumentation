# Getting Started with Development

Ready to contribute? Follow the setup instructions below to get your development environment ready.

## Prerequisites

- Node.js (v18 or higher)
- PostgreSQL
- Git

## Setup Steps

### 1. Fork and Clone

Fork the repository on GitHub, then clone your fork:

```bash
git clone https://github.com/YOUR_USERNAME/scrawn.git
cd scrawn
```

### 2. Install Dependencies

```bash
bun install
# or npm install / yarn install
```

### 3. Set Up Backend

Follow our [Quick Start Guide](/docs/scrawn/quick-start) to:
- Configure your database
- Set up environment variables
- Generate your API key

### 4. Run Development Server

```bash
bun run dev:backend
```

See [Backend Deployment](/docs/scrawn/backend-deployment#running-locally) for detailed local development setup.

## Development Workflow

1. **Create a Branch**: `git checkout -b feature/your-feature-name`
2. **Make Changes**: Write your code and test locally
3. **Run Tests**: `bun test` (ensure all tests pass)
4. **Commit**: Use clear, descriptive commit messages
5. **Push**: `git push origin feature/your-feature-name`
6. **Pull Request**: Open a PR with a clear description of your changes

## Project Structure

```
scrawn/
├── packages/
│   ├── backend/       # gRPC server (TypeScript)
│   └── sdk/           # Client SDK (TypeScript)
├── apps/
│   └── dashboard/     # Admin dashboard (Next.js)
└── docs/              # Documentation site
```

## Coding Standards

- Follow existing code style
- Write clear, descriptive commit messages
- Add tests for new features
- Update documentation as needed
- Ensure TypeScript types are properly defined
